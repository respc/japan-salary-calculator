# 个人事业主节税神器 - 产品企划文档

## 📋 项目概述

**项目名称：** 日本个人事业主节税优化平台
**目标：** 帮助日本个人事业主（個人事業主、フリーランス）最大化合法节税，提供智能化税务优化建议
**目标市场：** 日本约690万个人事业主和自由职业者

---

## 👥 目标用户画像

### 主要用户群体

**用户A：新手フリーランス**
- 年收：300-500万円
- 痛点：不了解税务制度，不知道如何合法节税
- 需求：简单易懂的节税指南，手把手教学

**用户B：成熟个人事业主**
- 年收：500-1000万円
- 痛点：知道一些节税方法，但不确定是否已最大化
- 需求：全面的额度追踪，确保不遗漏任何节税机会

**用户C：高收入自由职业者**
- 年收：1000万円+
- 痛点：税负过重，需要专业税务规划
- 需求：高级税务优化策略，税理士对接

## 🎯 Phase 1: MVP功能清单（2-3周开发）

### 1.1 用户模式切换
- [ ] 首页添加模式选择器
  - 会社員モード（现有功能）
  - 個人事業主モード（新功能）
- [ ] 选择后保存用户偏好（localStorage）
- [ ] 不同模式显示不同的UI和计算逻辑

### 1.2 个人事业主专用输入界面

**基本信息：**
- [ ] 事業収入（年額）
- [ ] 年龄
- [ ] 所在地区（都道府県）
- [ ] 家族构成（配偶者、扶养家属数）
- [ ] 申报类型（白色申告 / 青色申告）

**経費输入：**
- [ ] 経費合計
- [ ] 或详细経費分类输入
  - 仕入費
  - 租金・水电费
  - 通信费
  - 交通费
  - 交際費
  - 消耗品费
  - 其他

**已使用的控除输入：**
- [ ] 青色申告特別控除（10/55/65万円）
- [ ] 小規模企業共済（月额）
- [ ] iDeCo（月额）
- [ ] 生命保険料
- [ ] 地震保険料
- [ ] 医療費
- [ ] ふるさと納税
- [ ] 配偶者控除/扶养控除
- [ ] 青色事業専従者給与（如有）

### 1.3 核心计算逻辑

**税金计算：**
- [ ] 事業所得计算（収入 - 経費 - 青色控除）
- [ ] 所得税计算（累进税率）
- [ ] 住民税计算（10%）
- [ ] 国民健康保険计算（地域別料率）
- [ ] 国民年金（定额）
- [ ] 個人事業税（业种别）

**控除额度追踪：**
- [ ] 计算各项控除的已用额度
- [ ] 计算各项控除的剩余额度
- [ ] 计算潜在节税金额

### 1.4 节税建议系统（免费版：前3条）

**建议优先级算法：**
```
优先级 = 节税金额 / (实施难度 × 现金流影响)
```

**免费版显示：**
- [ ] Top 3 高优先级建议
- [ ] 每条建议包含：
  - 建议内容
  - 预计节税金额
  - 实施难度
  - 操作指南链接

**付费版显示：**
- [ ] 完整建议列表（10-15条）
- [ ] 详细实施步骤
- [ ] 相关资料下载
- [ ] 外部链接（affiliate）

### 1.5 结果展示

**手取り计算：**
```
事業収入
- 経費
- 青色申告特別控除
= 事業所得

- 各种控除（社会保险料、小規模共済等）
= 課税所得

- 所得税
- 住民税
- 国民健康保険
- 国民年金
- 個人事業税
= 手取り
```

**节税机会展示：**
- [ ] 当前税负分析
- [ ] 可优化项目列表
- [ ] 潜在节税总额
- [ ] 对比图表（当前 vs 优化后）
